# AI Voice Interview Platform

An interactive web application that helps users practice technical interviews through voice-based interaction. The platform uses Speech Synthesis and Speech Recognition APIs to simulate a real interview experience.

🌐 Live Demo



## Features

* Role-based interview questions
* Random question generation
* Text-to-Speech (Question Reading)
* Speech-to-Text (Answer Recognition)
* Interview Countdown Timer
* Dynamic User Interface
* Real-time Answer Display

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Web Speech API
* Speech Synthesis API

## How It Works

1. Select an interview role.
2. Click "Start Interview".
3. A random interview question is generated.
4. Use "Read Question" to hear the question.
5. Use "Start Answer" to answer using your voice.
6. Your speech is converted into text and displayed on the screen.
7. Complete the interview within the given time limit.
